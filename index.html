Covid 19 Global Data

Overview of Daily New cases and Deaths Globally

The Covid 19 has been the reason for the global pandemic since the beginning of 2020. There has been several phases of increase in cases as well as decline. This Narrative visualization gives an insight into the daily number of new cases and deaths across the globe, Continent and Country using an area chart.

Overview of Daily New cases and Deaths by Continent

This interactive page gives us an understanding of the total number of daily new cases and deaths recorded by Continent. This page allows the user to choose a Continent from the drop down list and displays the output accordingly in the form of an area chart.

Overview of Daily New cases and Deaths by Country

This interactive page gives us an understanding of the total number of daily new cases and deaths recorded by Country. This page allows the user to choose a Country from the drop down list and displays the output accordingly in the form of an area chart.
